# MoDesign — Website Iklan Jasa Desain

## Cara pakai
1. Buka `index.html` di browser (double click).
2. Ubah nomor WhatsApp di `index.html` pada link `id="waRaw"` (bagian “Kontak cepat”).
3. Ubah email / nama brand (jika perlu) di `index.html` (cari `mailto:` dan teks “MoDesign”).
4. Logo brand otomatis ambil dari `embed/mogaps-logo.png` (jika tidak ada, fallback ke `embed/mogaps-logo.jpg`).
5. Preview slider: `embed/sertifikat.png` (Sertifikat), `embed/poster.png` (Poster), `embed/polaroid.png` (Polaroid).

## Struktur file
- `index.html` — halaman utama
- `minecraft.html` — halaman order 3D JSON Minecraft
- `styles.css` — styling
- `minecraft.css` — styling ala Minecraft
- `main.js` — tombol WhatsApp + form order
- `favicon.svg` — ikon tab
